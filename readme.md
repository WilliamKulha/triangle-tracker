# _Triangle Tracker_

#### _A dummy webpage that will tell a user (based on their input) whether the sides add up to a triangle and what kind of triangle it is, 9-15-2018_

#### By _**William Kulha**_

## Description

_This is just a dummy webpage that I made to practice javascript branching statements through learnhowtoprogram.com._

## Setup/Installation Requirements

* _Download or clone this repo <code>$git clone https://github.com/zangiboy/triangle-tracker.git</code>_
* _Navigate to the folder where you saved this repository._
* _Open index.html with the browser of your choice and enjoy_
